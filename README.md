# automapshot-web

> Automate screenshots using [FTE](https://fte.triptohell.info/) (web) and [Playwright](https://playwright.dev/)

## Setup

```shell
yarn install
```

* Rename `config.example.ts` to `config.ts`
* Rename `config.maps.example.ts` to `config.maps.ts`

## Usage

```shell
npx playwright test
```
